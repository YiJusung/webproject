# Hourly Pulse Frontend

Hourly Pulse의 React 웹 인터페이스입니다.

## 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행 (http://localhost:3000)
npm start

# 프로덕션 빌드
npm run build
```

## 환경 변수

`.env` 파일을 생성하여 API URL을 설정할 수 있습니다:

```
REACT_APP_API_URL=http://localhost:8000/api
```

## 기능

- 📊 실시간 통계 대시보드
- 🏆 이슈 랭킹 표시
- 🤖 AI 분석 결과 표시
- 📰 최근 수집 데이터 표시
- 🔄 30초마다 자동 새로고침
